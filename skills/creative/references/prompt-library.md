# Prompt Library — Joshua Bryan Maddox Creative Assets

## Joshua's Fixed Brand DNA Block

Include this verbatim at the start of EVERY image prompt. Do not paraphrase. Do not shorten.

```
Shot on Sony A7 IV, 35mm f/1.8, Kodak Portra 400 color grade. No airbrushing. No oversaturation. No symmetrical composition. Not stock photography. Not CGI. Not a digital render. Natural grain. Editorial documentary aesthetic.
```

## Prompt Swap Guide

When adapting templates, change only the **variable content block**. Never change the fixed brand DNA block.

**Variables you can swap:**
- Time of day: `blue hour` → `late afternoon golden hour` → `early morning`
- Light source: `desk lamp amber` → `monitor glow blue` → `north window diffuse`
- Subject: `laptop and notebook` → `espresso and sketchbook` → `multiple screens`
- Camera move: `slow dolly in` → `static` → `slow pan left`
- Duration: `6 seconds` → `8 seconds`

**Variables you must NOT change:**
- Camera body: always Sony A7 IV
- Lens: always 35mm f/1.8 (adjust aperture only for depth of field intent)
- Film stock: always Kodak Portra 400 (use Cinestill 800T only for night/neon scenes)
- Style prohibitions: always include the "not stock photography, not CGI" block
