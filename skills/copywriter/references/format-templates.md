# Format Templates

Structural templates for each content format. Follow these exactly when
drafting content. These are not suggestions -- they are the format grammar.

---

## LinkedIn Carousel (8-10 slides)

**Slide 1: Hook Slide**
- Bold statement or provocative question.
- No more than 15 words.
- This is the thumbnail -- it must stop the scroll on its own.
- No branding clutter. Just the hook.

**Slide 2: Context / Problem**
- Why this matters right now.
- 2-3 short sentences max.
- Set up the tension that the rest of the carousel resolves.

**Slides 3-8: Core Content (one idea per slide)**
- One single idea per slide. No exceptions.
- Use numbered lists, before/after comparisons, or framework diagrams.
- Keep text under 50 words per slide.
- Each slide should be comprehensible without reading the others.
- Use consistent visual structure across these slides.

**Slide 9 (or second-to-last): Summary**
- The key takeaway in one sentence.
- This is the "if you remember nothing else, remember this" slide.

**Final Slide: CTA**
- "Follow [name] for more on [pillar topic]"
- Include a comment prompt: a question or "Save this for later."
- Keep it clean -- do not overload with multiple CTAs.

**Design Notes:**
- Clean, minimal. White space is your friend.
- Dark backgrounds with light text OR light backgrounds with dark text.
- One visual element per slide (icon, diagram, number).
- Brand-consistent color palette throughout.
- PDF upload preferred over image series.
- Dimensions: 1080x1080px (square) or 1080x1350px (portrait).
- Max 20 slides per carousel (platform limit).

## LinkedIn Text Post (150-300 words)

```
[Hook line -- under 210 characters. This is what shows before "see more".]

[Empty line for visual break]

[Paragraph 1: Context or story setup. 2-3 sentences.]

[Paragraph 2: The insight, framework, or lesson. 2-3 sentences.]

[Paragraph 3: Application or evidence. 2-3 sentences.]

[Closing: Question to the audience OR actionable takeaway. 1-2 sentences.]

#Hashtag1 #Hashtag2 #Hashtag3
```

**Rules:**
- First line under 210 characters (truncation point before "see more").
- Empty line after hook creates visual break on mobile.
- Short paragraphs: 2-3 sentences each. Use line breaks between them.
- Bold key phrases with **asterisks** where appropriate.
- No emojis in the body unless it is an ask/share post.
- 3-5 hashtags on a separate final line.
- Total length: 150-300 words.

## LinkedIn Newsletter (800-1200 words)

```
Subject: [Under 60 characters. Use number, question, or contrarian framing.]

[Opening hook paragraph: 2-3 sentences. State what the reader will learn.]

## [Section 1 Heading: The Context / Problem]
[200-300 words. Set up the situation.]

## [Section 2 Heading: The Framework / Insight]
[200-300 words. Deliver the core idea.]

## [Section 3 Heading: The Application / Examples]
[200-300 words. Show it in practice.]

## What This Means For You
[100-150 words. Summary + CTA (follow, comment, share).]
```

**Rules:**
- Subject line under 60 characters.
- 3-5 subheaded sections.
- Each section can stand somewhat independently.
- Can embed 1-2 data visualizations or images.
- Closing CTA should be specific: ask a question, prompt a share, or direct to follow.
- Total length: 800-1200 words.

## X Thread (7-12 tweets)

```
1/ [Self-contained hook. Must work as a standalone tweet. End with thread indicator.]

2/ [Context or problem setup. One complete thought.]

3/ [First key point. One idea, fully expressed.]

4/ [Second key point. Transitional phrase at start.]

5/ [Third key point or supporting evidence.]

6/ [Fourth key point or story/example.]

...continue as needed...

[N-1]/ [Summary of the thread in one tweet. "The bottom line:"]

[N]/ [CTA: "If you found this useful, follow me for more on [topic]. RT tweet 1 to share."]
```

**Rules:**
- Tweet 1 must be self-contained and hook-worthy on its own.
- Number each tweet: "1/" "2/" etc. at the start.
- One complete idea per tweet. Each comprehensible without the others.
- Use transitional phrases: "Here's the thing:", "But here's what most miss:", "The result?"
- Max 280 characters per tweet. Count carefully.
- 7-12 tweets total. Shorter threads outperform longer ones.
- Penultimate tweet summarizes. Final tweet is always a CTA.

## X Single Tweet (280 characters max)

```
[One punchy idea. One or two short sentences. Hashtag at end.]
```

**Formats that work:**
- Hot take: "[Contrarian opinion]. And here's why."
- Data point + reaction: "[Stat]. Let that sink in."
- Question: "[Observation]? What's your take?"
- Observation: "Nobody talks about [X] enough."
- Contrarian: "Unpopular opinion: [statement]."

**Rules:**
- 280 characters max. Count every character.
- 1-2 hashtags max, at the end.
- No links in the main tweet (put links in a reply).
- No emojis unless they replace a word or add genuine meaning.
